# Design Patterns em Java

## Descrição
Este projeto foi desenvolvido como atividade da disciplina de Java com o objetivo de demonstrar a aplicação de padrões de projeto (Design Patterns).

## Padrões utilizados
- Singleton
- Factory
- Strategy

## Objetivo
Demonstrar conceitos de reutilização de código, organização e boas práticas de desenvolvimento utilizando Java.
